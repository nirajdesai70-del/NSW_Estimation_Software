# Design Documents

**Purpose:** All design documents organized by component

## Subfolders

- **FEEDER_BOM/** - Feeder BOM design and execution documents
- **MASTER_BOM/** - Master BOM backend design documents
- **PANEL_BOM/** - Panel BOM planning and design documents
- **GENERIC_ITEM_MASTER/** - Generic Item Master design documents
- **SPECIFIC_ITEM_MASTER/** - Specific Item Master design documents
- **PROPOSAL_BOM/** - Proposal BOM design documents
- **RESOLUTION_B/** - Resolution B design and rules

## Quick Start

1. Navigate to component-specific subfolder
2. Check design index or master index in each subfolder
3. Review design documents as needed
