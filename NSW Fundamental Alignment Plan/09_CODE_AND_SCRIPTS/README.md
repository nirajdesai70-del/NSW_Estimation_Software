# Code and Scripts

**Purpose:** Implementation code, scripts, SQL, and migrations

## Subfolders

- **PHP_SERVICES/** - PHP service files (BomEngine, BomHistoryService)
- **PHP_CONTROLLERS/** - PHP controller examples
- **PHP_MIGRATIONS/** - Database migrations
- **SQL_SCRIPTS/** - SQL verification and execution scripts
- **SHELL_SCRIPTS/** - Shell automation scripts
- **PYTHON_SCRIPTS/** - Python governance automation scripts

## Quick Start

1. Navigate to script type subfolder
2. Review README files in each subfolder
3. Use scripts as needed for execution
