## üéØ Review Methodology

### For Each File:

1. **Read the File**
   - Understand its purpose
   - Identify all sections
   - Note key information

2. **Document Content**
   - List all sections
   - Extract key information
   - Note important details

3. **Map to Target**
   - Determine target location
   - Determine target file
   - Note if content needs splitting

4. **Check References**
   - List all internal links
   - List all cross-file references
   - Plan link updates

5. **Verify Completeness**
   - Ensure nothing missed
   - Ensure nothing lost
   - Ensure proper mapping
